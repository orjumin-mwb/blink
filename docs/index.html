BLINK API

Next-Generation URL Security Analysis Platform

Overview

The BLINK API provides comprehensive URL security analysis with real-time threat detection, AI-powered verdicts, and detailed security assessments. Built with Go microservices architecture for high performance and scalability.

Base URL: http://localhost:8080 (configurable via PORT environment variable)

Quick Start Test

curl -X POST http://localhost:8080/check -H "Content-Type: application/json" -d '{"url":"https://example.com"}'

API Endpoints

Complete reference of available endpoints and their capabilities

Method Endpoint Purpose Content-Type Real-time
GET /health Service health check application/json No
POST /check Basic URL validation & analysis application/json No
POST /deep-check Comprehensive security analysis application/json or text/event-stream Yes (SSE)
POST /screenshot Website screenshot capture application/json No
GET /ui Interactive web interface text/html No
GET /ui/stream Real-time UI updates text/event-stream Yes (SSE)
GET /screenshots/{date}/{filename} Static screenshot files image/jpeg or image/png No

Key Features

  • Real-time Analysis: Server-Sent Events for live security assessment
  • AI-Powered Verdicts: ScamGuard integration for threat intelligence
  • Comprehensive Scanning: DNS, TLS, content, and API analysis
  • Visual Verification: Screenshot capture with BlurHash encoding
  • Technology Detection: Framework and library identification
  • Security Scoring: Numerical trust ratings with detailed explanations
  • Privacy Assessment: Tracking and fingerprinting detection
  • Payment API Detection: E-commerce platform identification

Performance Targets

  • Basic Check: Sub-2 second response time
  • Deep Analysis: Sub-5 second response time
  • Screenshot Capture: Sub-10 second response time
  • Real-time Updates: <100ms event latency
  • Concurrent Analysis: Up to 5 simultaneous deep checks

Rate Limits

  • Basic Checks: 100 requests per minute per IP
  • Deep Analysis: 20 requests per minute per IP
  • Screenshots: 10 requests per minute per IP
